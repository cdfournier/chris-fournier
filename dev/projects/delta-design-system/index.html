The Delta Design System Project Details

Deliverables

UX design, HTML, SCSS/CSS, jQuery, and Sketch

I create bespoke, responsive websites and applications — both for a living and for fun. I frequently find myself reusing my own code, so I decided to build The Delta Design System, an open-source system that makes it easy to design responsive websites, apps, and prototypes with code that is semantic, flexible, customizable, and wicked efficient.

Audience

Anyone looking for a clean, and simple way to build prototypes or production-ready sites and applications, but without the complicated learning curve of other framework solutions.

Ethan Marcotte’s Responsive Web Design article and Brad Frost’s Atomic Design methodology, as well as flexbox and Sass are some of the things that inspired my decision to build a design system. Responsive web design, in particular, changed my life. I love everything about the concept of responsive design.

I wanted to create a css and html framework that I could use to jump start my projects. I made this project open-source to encourage others to iterate on my work.

Outcome

I created what I consider to be a robust and complete user experience design resource, delivering something that was open-source with usable code, and a static design asset in the form of a Sketch library.

Visit The Delta Design System Github repo or online guide for more information. And, if you’re looking for more of my work, check out the demo site I created using The Delta Design System.
