Introduction to The Delta Design System

Personal ProjectThe Delta Design System

An open-source design system that can be used to build responsive sites, apps, and prototypes quickly.

The Delta Design System project details

Deliverables

UX design, HTML, SCSS/CSS, jQuery, and Sketch

I create handcrafted, responsive websites and applications for a living and for fun, and I frequently find myself reusing my own code. So I decided to turn my work into an open source design system that makes it easy to design responsive websites, apps, and prototypes with code that is semantic, flexible, customizable, and wicked efficient.

Audience

Anyone looking for a quick, clean, and easy way to build prototypes or production-ready sites and applications, but without the fast food approach of other framework solutions.

I’ve been designing for the web for… well, let’s just say I’ve been designing for the web for awhile now. At some point in my early history, someone said that – if you really want to know what you’re doing – you need to design in html and css. Some other things that inspired me include Ethan Marcotte’s Responsive Web Design article, flexbox, and Sass.

Working almost hand-in-hand with responsive design, flexbox was a concept I was able to get behind immediately. I may not have a complete grasp on flexbox, but the concept and the syntax was easy enough to change the way I thought about code and design.

Lastly, it took me a long time to ramp up to Sass, but once I was able to start using the basics, it had a profound effect on the way I went about my work.

Ethan Marcotte’s Responsive Web Design article changed my life. I love everything about the concept of responsive design.

I’ve been in more than a few situations as a user experience designer where the approach taken and tools used were both lacking and sloppy. I wanted to create a viable html and css framework that I could use to jump start my projects. I was also hoping to provide a resource for others and maybe even get others to add their input and expertise.

In order to create what I considered to be a robust and complete user experience design resource, I needed to deliver something that was open source and included usable code, as well as a static design asset like a Sketch library. The intent was to provide something simple, flexible, and at least attempts to meet accessibility standards.

Visit The Delta Design System Github repo or online guide for more information. And, if you’re looking for more of my work, check out the demo site I created using The Delta Design System.
