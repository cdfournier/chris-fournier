Summary

I love to create responsive web sites and applications. I frequently find myself reusing my own code, so I decided to build The Delta Design System, an open-source system that makes it easy to design responsive web sites, apps, and prototypes with code that is semantic, flexible, customizable, and wicked efficient.

I wanted to create a smart and mindful scss and html framework that anyone could use to jump start projects. I made this project open-source in hopes that others might iterate on my work.

Visit The Delta Design System Github repo or online guide for more information. And, also be sure to check out the demo site I created using The Delta Design System.
